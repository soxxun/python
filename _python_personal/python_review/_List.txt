List
 - 데이터의 집합
 - 여러개의 데이터들을 저장하고 관리하는 형태
 - 데이터의 변경이 가능

 *선언 형식
   - []을 통해 선언
   - 변수명 = [데이터, 데이터, 데이터...]

 *호출 방식
   - 인덱싱과 슬라이싱 모두 사용 가능
   - 변수명[인덱스]
	- index : 0번째부터 저장된 데이터 개수보다 -1

list에서 자주 사용되는 함수
1) append(value) : 리스의 끝에 값을 추가하는 함수

2) iter : 다량의 데이터를 가지고 있는 데이터의 형식... list, tuple, dic, str
   extend(iter) : 리스트 끝에 list, tuple, dic, str의 값을 하나씩 추가하는 함수
	ex. (5,6) -> 5, 6 한 개씩 데이터를 뽑아서 저장

3) insert(idx,value) : 특정 위치에 값을 추가하는 함수로, 인덱스 범위를 벗어나면 제일 끝에 데이터를 추가함

4) pop : 기본적으로 마지막 인덱스 안에 있는 값을 반환하고 삭제하는 함수 (인덱스를 적지 않을 때 : pop() )
           인덱스 값을 적으면 인덱스 위치의 데이터를 삭제
           인덱스 범위를 벗어나면 error

5) remove(value) : 특정 값에 해당되는 것을 찾아 삭제하는 함수
                        대신 값이 없으면 error

6) clear() : 모든 값을 삭제하여 빈 리스트만 남기는 함수

7) count(value) : 리스트에서 일치하는 값의 수를 반환하는 함수

8) index(value) :  리스트에서 일치하는 값의 인덱스 번호를 반환하는 함수

9) reverse() : 리스트의 모든 값을 뒤집어서 나열하는 함수

10) sort(reverse = True/False) : 리스트의 값을 오름차순(False) 생략 가능 (내림차순 Ture)
