객체(object)
 - 어떠한 대상을 표현하기 위한 정보(변수)와 그 대상을 컨트롤 하기 위한 기능(함수)을
   하나의 루틴으로 묶여있는 개별적인 개체로 표현되는 변수
ex) 회원가입 후 정보를 저장하는 것이 있고 로그인 시 아이디를 찾는 함수의 기능이 있을 것

 - class라는 사용자 정의 자료형으로 만들어진 변수
 - Python에서 모든 데이터 타입을 'object'라고 함

Tuple(튜플)
 - 데이터들의 집합
 - 여러 개의 데이터들을 저장하고 관리하는 형태
 - 데이터 변경 불가능 ***
 - 같은 형태와 같은 성격의 데이터들을 보다 쉽게 관리할 수 있도록 사용

 *선언형식
 - ()를 통해 선언
 - () 생략 가능
   (변수처럼 하나의 공간을 만드는 것이 아니라 여러 개의 변수를 만드는 것)

 *호출
 - 인덱싱, 슬라이싱을 통해 데이터 공간 호출

Packing - 자동으로 Tuple타입으로 저장시키는 것 ... [ () 생략 ]
Unpacking - 자동으로 Tuple타입의 데이터를 변수 타입으로 저장시키는 것

Tuple에서 사용되는 함수들

count(value)
 - 튜플에서 일치하는 값의 수를 반환하는 함수
 - 없으면 0 반환, 있으면 갯수 반환
   (ex. 튜플에서 3을 저장했으면 3이 몇 개 있는지를 출력)

index(value)
 - 튜플에서 일치하는 값의 인덱스 번호를 반환하는 함수
 - 데이터가 없으면 error
   (ex. 튜플에 1, 2, 3을 'tp'라는 변수명으로 저장해 놓고 tp.index(2)를 호출하면 인덱스 2번의 값 출력
