Dictionary(사전형)
 - 데이터의 집합
 - key와 value값의 형태로 1:1쌍으로 저장하여 사용한다.
 - key값은 중복 불가, 변경 불가
 - value값은 중복 허용, 변경 가능

선언
 - {}을 통하여 선언
 - key와 value를 구별하기 위해 중간에 :을 적어줌
	변수명 = {key : value, key : value, key : value}

호출
 - key를 통하여 호출
 - 형식
	변수명[key]

만약 과일을 검색
 - "과일에는 ㅇㅇ, ㅇㅇ, ㅇㅇ가 있다"라고 할 때, 그 값을 value라고 하며 여기서 '과일'이 key값
 - 비밀번호 찾기 할 때 ID를 통하여 PW를 찾으므로 ID가 key값, PW가 value값이라고 보면 됨
